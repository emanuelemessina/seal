# SEAL
 Structured Encoder for Ancient Logograms

## Requirements

```
pip install -r requirements.txt
```

Install [pytorch](https://pytorch.org/get-started/locally/) for the specific CUDA version you have. 

## Train

```
cd model
python faster-rcnn.py
```
